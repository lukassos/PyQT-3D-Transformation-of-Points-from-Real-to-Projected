Ako rozbehat transformacny softik (FREE a Licencovane)

  1. Download Python For Windows

    Podla typu procesora si vyber verziu SW (https://www.python.org/)

    • Python 64 bit
    • Python 32 bit

  2. Install Python

    Tento krok bude asi chvilku trvat tak po spusteni si kludne daj robit caj

  3. Download PyQT5 For Windows

    Podla typu procesora si vyber verziu SW (https://wiki.python.org/moin/PyQt)

    • PyQt5 64 bit
    • PyQt5 32 bit

  4. Install PyQT5

    Kludne si mozes ist ten caj zaliat :D dat ho na balkon vychladnut

  5. Nastavenie systemu a doinstalovanie matematickeho balika NumPy

    Na dalsie kroky mame skript install_ver34.bat, spusti ho bez admin prav. Ak nepojde dvoj-klikom tak cez konzolu (klavesova skratka "Win+R", potom cmd.exe, potom klik OK).

    Chystame sa spustit tieto prikazy

    setx path "%PATH%;C:\Python34;C:\Python34\Scripts;C:\Python34\Lib\site-packages\PyQt5"
    assoc .py=Python.File
    ftype Python.File=C:\Python34\pythonw.exe "%1" %*
    pip install numpy

  6. Spustenie programu

    Program sa spysta cez start.py, co vyvola Qt5 generovane okno (s malymi upravami to je kompatibilne aj s linuxom ;-) )

  7. Praca s programom

    • Vstupy

      Vstupom su dva textove subory s formatom "ID_bodu X_suradnica Y_suradnica Z_suradnica"(oddelovacom je jedna alebo viac medzier/tabulatorov, takyto subor mozno vygenerovat aj z Excelu). Jeden subor predstavuje predlohu (v programe ako "Predloha")a druhy namerane udaje v realite (v programe ako "Meranie"). Vstupne subory Vstupy mozno najst cez tlacitka "Hladat" alebo iba nakopirovat vhodne formatovany vstup do textovych casti im prisluchajucih. Textove casti su interaktivne s podporou historie, drag and drop a aktivne obnovuju tabulku pre vyber Zachytnych bodov. Ak su do textovych casti prenesene subory z prehliadaca suborov vo Windows, adresa je nacitana ako textovy subor (to ta drag and drop funkcia).

    • Zachytne body

      Zachytne body sluzia na vypocet transformacie medzi priestormi bodov Predlohy a Merania. Body su v tabulke automaticky zoradene podla ID. Zobrazene body prisluchaju bodom z merania. Vyberom riadkov (Tlacidlo "Vyber vsetky" alebo po jednom mysou pri drzani ctrl "Ctrl+LeftClick") v tabulke sa urci, ktore body zo spolocnych budu pouzite ako Zachytne. Minimalny pocet Zachytnych bodov je #ZATIAL NEZNAMY .. UVIDIME PO VYPOCTOCH A TESTOCH#. Ak su nahodou v subore Merania chybne udaje (ine ako cisla) program danu bunku vysvieti na cerveno. Ak su v tabulke dva riadky vzajomnym skalarnym nasobkom budu oznacene rovnakou farbou (Ak je farba podobna vo viacerych pripadoch, stlac "Obnovit", farba sa nahodne meni).

      Zachytne body mozno menit v tabulke priamo, zmeny sa odrazia na VYPOCTE ale nie na vstupnom subore.

    • Vypocet

      Spusta sa tlacitkom "Prepocitat" a vystupom je subor v rovnakom formate ako boli vstupy

      Zachytne body mozno menit v tabulke priamo, zmeny sa odrazia na VYPOCTE ale nie na vstupnom subore.
